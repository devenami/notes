## 查看占用端口的 PID

打开命令行输入`netstat -ano  | findstr "端口号"`

```
协议		本地地址			外部地址		状态			PID
TCP		0.0.0.0:查询端口号	0.0.0.0:0		LISTENING	pid号码
```



## 查看占用端口的进程

命令行输入`tasklist | findstr "pid号码"`

```
进程名称	pid		....
```



## 根据PID终止进程

打开任务管理器 -> 进程页面，

点击查看 -> 选择列，在窗口中将PID勾上。

在进程页面中查找指定的PID，如果找不到，将左下角的**显示所有用户的进程**打上勾。

选中指定的PID，右键选择结束进程。

或者在CMD命令中执行：`taskkill /f /t /im 进程名`
